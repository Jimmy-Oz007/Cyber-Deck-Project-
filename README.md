# Cyber-Deck-Project-
Raspberry Pi Cyber Deck inspired by retro techs.

Project #1 — Raspberry Pi Cyberdeck

Overview

This project is the first build in my “100 IT Projects” challenge.

The goal of this project is to create a portable cyberdeck inspired by retro typewriters and cyberpunk terminal systems using a Raspberry Pi 5 and Linux.

The cyberdeck is designed as a compact portable computing platform for:

* Linux learning
* Python programming
* system monitoring
* networking experiments
* future cybersecurity and AI projects
